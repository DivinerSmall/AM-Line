# AM-Line — Anti Messenger

AM-Line — экспериментальный текстовый чат **без истории сообщений**.  
У каждого участника есть своё поле ввода, и в реальном времени видно, что он печатает. 
После окончания сессии история не сохраняется.

## Статус проекта

Сейчас проект находится на стадии **идеи и спецификации** (MVP ещё не реализован).

## Основная идея

- Нет классической ленты сообщений.
- У каждого участника одно текущее поле текста.
- Другие участники видят этот текст в реальном времени.
- Данные комнаты живут только пока идёт сессия.

## Сценарии использования

- Приватные разговоры “без следов”.
- Сессии терапии / коучинга / честных разговоров.
- Экспериментальные форматы общения, брейнштормы, арт-проекты.
